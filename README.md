# ambig-turn-bench: Ambiguous Conversational Q&A Evaluation Suite

A comprehensive evaluation framework for conversational question answering systems, inspired by [CoIR](https://github.com/CoIR-team/coir) and compatible with PyTerrier RAG pipelines.
